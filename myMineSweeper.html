Instructions

Each square is either a bomb or not. If it is not a bomb when it is uncovered it will displayed the number of bombs next to it. The game ends when you have uncovered all of the not bombs. If you right click it becomes a flag. If you left click it uncovers the square. If you uncover a bomb you lose.
